MOTION LAB // MECHANISM STUDY

正切机构｜工程化应用案例

正切机构通过定垂距接触获得角度—线位移转换，可用于有限角范围内的精密调角、转位与光学读数。

MECHANISM LOGIC / 机构链路
直线调节输入 → 正切机构 → 精密角位移 → 光学读数或轮廓扫描姿态
CASE 01 // POLARIMETER

旋光仪检偏器调角

旋光仪可采用正弦机构或正切机构，将检偏器旋转角度转化为易读取的线位移，从而表示被测物质旋光度。

执行端：检偏器调角
CASE 02 // METROLOGY

大型非球面轮廓扫描测量

轮廓扫描装置中，正切机构作为转位角发生装置，配合直线导轨与精密角度检测，实现曲面扫描姿态控制。

执行端：转位角模块

工程器械与衍生类型

光学读数器械旋光仪检偏器角度读取
精密测量装备球面/非球面轮廓扫描转位
航空仪表传动正切与双正切角度耦合

调研依据

  1. CN2173960Y《Polarimeter》：正弦或正切机构驱动检偏器并显示旋光角。 资料入口
  2. CN103105141A：大型球面非球面轮廓扫描装置中转位角发生装置设计为正切机构。 资料入口
  3. E. H. J. Pallett, Aircraft Instruments：正切与双正切杆式传动说明。 资料入口
